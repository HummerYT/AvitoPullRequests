# AvitoPullRequest Service

Микросервис для управления Pull Request системами с назначением ревьюверов, статистикой и командами.

1. Чтобы запустить сервис, используйте `make up` (чтобы завершить - `make down`)
2. Чтобы пересобрать сервис, используйте `make rebuild`
3. Чтобы запустить нагрузочное тестирование, используйте `make load-test`
4. Чтобы просмотреть логи, используйте `make logs`
5. Чтобы посмотреть статус контейнера, используйте `make status`

## Функциональность

- **Управление командами** - создание команд и участников
- **Управление пользователями** - активация/деактивация пользователей
- **Pull Requests** - создание, мерж, переназначение ревьюверов
- **Статистика** - общая статистика по PR, пользователям и командам
- **Автоматическое назначение** - случайное распределение ревьюверов внутри команды

## API Endpoints:

### Health Check
- `GET /health` - проверка работоспособности сервиса

### Teams
- `POST /team/add` - создание команды
- `GET /team/get?team_name=name` - получение информации о команде

### Users
- `POST /users/setIsActive` - активация/деактивация пользователя
- `GET /users/getReview?user_id=id` - получение PR для ревью пользователя

### Pull Requests
- `POST /pullrequest/create` - создание PR
- `POST /pullrequest/merge` - мерж PR
- `POST /pullrequest/reassign` - переназначение ревьювера

### Statistics
- `GET /stats` - получение статистики

## Используемые технологии

- **Go 1.24** - основной язык
- **PostgreSQL 15** - база данных
- **Docker & Docker Compose** - контейнеризация
- **Make** - автоматизация задач
